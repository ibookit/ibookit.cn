NVIDIA Jetson Nano Developer Kit 新手入门 (3)

安装 OpenCV

  • 暂时参考 https://www.pyimagesearch.com/2018/08/15/how-to-install-opencv-4-on-ubuntu/ ，还没有找到专门针对 Jetson Nano 的安装文档

    • 安装一些工具
    • 安装一些图像图像库
    • 安装 GTK
    • 安装 Python 开发包
    • 下载 OpenCV 源代码
    • 创建一个新的 Python 虚拟环境，并进入这个虚拟环境
    • 安装 NumPy
    • 准备编译 OpenCV
    • 开始编译，需要大约 60 分钟
    • OpenCV 编译完成后，将以下库文件改名，并在虚拟环境下建立一个符号链接
    • 验证 OpenCV 4 安装是否成功
    • 安装 dlib
      • 安装 dlib 过程中使用 htop 经常看到两个 CPU 在同时做编译，散热片也很烫，约 20 分钟后卡死。
      • 加个风扇，重启机器，重新 install dlib，用了 29 分钟
    • 安装 face_recognition

  • 安装 imutils

Demo

参考 Marcelo Rovai’s Post: Real-Time Face Recognition: An End-to-End Project

  • 首先下载代码
  • 进入 cv 虚拟环境

  • 修改以下文件，将文件中 img = cv2.flip(img, -1) # flip video image vertically 注释掉。Marcelo Roval 的文章中所使用的设备是 Rampberry Pi 3B，可能需要将摄像头图像倒置，我使用 Jetson Nano 时发现如果不把这一行注释掉，摄像头图像就是反的

    • 01_face_dataset.py
    • 02_face_training.py
    • 03_face_recognition.py

  • 采集人脸，建立 dataset，采集时需要面对摄像头，共采集 30 张照片

  • 采集时会提示输入人脸顺序号，可以输入 1，代表 1 号人物
  • 训练
  • 为 1 号人物改名
    • 修改 03_face_recognition.py, 将第 26 行中数组的第 2 个名字改为想用的名字 (现在是代码作者的名字 Marcelo)
  • 开始识别

这时摄像头窗口出现，显示摄像头采集的图像，需要将脸靠近摄像头，视频窗口中被识别出的脸会被矩形框住，并显示名字
